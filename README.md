# 🌍 Language Translator – Python CLI Application

A command-line based Language Translator built using **Python 3.11** and the **Googletrans** library.  
This application allows users to translate text into multiple languages with automatic source language detection.

---

## 🚀 Features

- 🌐 Translate text into 15+ languages
- 🔎 Automatic source language detection
- 🔊 Pronunciation support (if available)
- 🧠 Continuous translation mode
- ❌ Graceful error handling
- 💻 Lightweight CLI interface

---

## 🛠 Tech Stack

- **Python 3.11**
- **Googletrans (4.0.0-rc1)**
- Command Line Interface (CLI)

---

## 📸 Sample Usage
Enter language code: hi
Enter text: Hello world

Translated Text: नमस्ते दुनिया
Detected Source Language: en
